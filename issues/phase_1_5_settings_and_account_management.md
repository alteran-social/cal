**Goal**: Users can manage their account, integrations, and preferences.

### PRD 1.5 - User Settings

**User Stories**
- As a user, I can edit my booking page display name and description
- As a user, I can manage connected calendar accounts
- As a user, I can set notification preferences
- As a user, I can delete my account and all associated data

**Deliverables**
- Settings dashboard
- Profile customization (display name, bio, profile photo from atproto)
- Connected accounts management (disconnect Google)
- Notification preferences (email on/off per type)
- Data export (GDPR compliance)
- Account deletion flow (removes D1 data, preserves atproto records)
